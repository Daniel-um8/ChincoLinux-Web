Instalación

Live USB o Máquina Virtual

Arranca desde un pendrive sin instalar nada, o pruébalo en una VM como VirtualBox, VMware o GNOME Boxes.

Paso 1 — Descarga la ISO

Obtén la imagen desde el repositorio oficial.

Descargar ISO

Paso 2 — Crea un Live USB

Graba la .iso en un pendrive (≥4GB) con tu herramienta favorita.

Rufus (Windows) balenaEtcher

Paso 3 — Arranca desde USB

Reinicia tu equipo, entra al menú de arranque y selecciona el pendrive como dispositivo de inicio.

Opción — Máquina Virtual

Si prefieres, monta la ISO en una VM.

VirtualBox GNOME Boxes

Requisitos sugeridos

CPU x64, 4GB RAM (8GB ideal), 16GB de espacio libre, pendrive ≥4GB para Live USB.

Consejos útiles

Si no ves el USB en el arranque, revisa el modo UEFI/Legacy o el “Secure Boot”.
